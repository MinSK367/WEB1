HTML란?

HTML (Hypertext Markup Language,하이퍼텍스트 마크업 언어)는
우리가 보는 웹페이지가 어떻게 구조화되어 있는지 브라우저를 통해 알 수 있도록 하는 마크업 언어입니다.
프로그래밍 언어는 아니며, 개발자는 HTML을 이용해 복잡하게, 또는 간단하게도 프로그래밍 할 수 있습니다.
HTML은 elements로 구성되어 있으며, 이들을 적절한 방법으로 나타내기 위해 각 컨텐츠의 여러 부분들을 감싸고 마크업 합니다.
tags 는 웹 상의 다른 페이지로 이동하게 하는 하이퍼링크 내용들을 생성하거나, 단어를 강조하는 등의 역할을 합니다.
